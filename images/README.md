# Images Folder

Add your property photos here. See CONTENT-PLAN.md for the full list of photos needed.

## Naming Convention
Use lowercase, hyphens, no spaces:
- `hero-building-exterior.jpg`
- `west-storefront-interior.jpg`
- `black-river-waterfront.jpg`
- `great-hall-event-space.jpg`

## Recommended Size
- Width: 1600px
- Format: .jpg for photos
- Max file size: 500KB per image
- Use https://squoosh.app to compress

## Required for Open Graph (Social Sharing)
Create an `og-image.jpg` at 1200x630 pixels — this is what shows when the website link is shared on Facebook, LinkedIn, or Twitter.
